Periodic Table Selector

A WPF based selector control in the shape of the Periodic Table of Elements

Download .zip Download .tar.gz View on GitHub

What It Is

The Periodic Table Selector is a WPF control where the Chemical Elements can be selected. Since the Periodic Table and the Chemical Elements both happen to be WPF custom-controls, they can be re-styled and re-templated. By default, selecting a Chemical Element will activate a glow effect around the text and on the inside of the border. The color of the glow effect, the color of the foreground, and the color of the background can be set on a case by case basis on each Chemical Element. This might not sound very impressive, but people have been caught playing around with this for no other reason than to play with it.

Since the Periodic Table control is a selector, a developer can bind to the selected Chemical Elements. This list of selected Chemical Elements can be changed by changing the selected state of a Chemical Element or add/removing a Chemical Element to the selected items list. The selected state of the Chemical Elements are updated when the list changes.

What It's For

It's a selector control for selecting chemical elements. You can use it anyway you want. If you want to create a game where the person has to select the correct chemical element to answer a question, then you can use if for that. If you need to create an image of a Periodic Table, but need to changes a few minor details, it's good for that. You can even use it to create heatmaps of the chemical elements.
